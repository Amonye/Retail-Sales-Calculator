Accounting & Sales Management System (Python CLI Project)

Project Overview

This project is a Python-based Accounting Management System designed to track and manage daily business operations such as:

Shift sales (morning/evening)
Employee working hours
Salary calculations
Daily expenditures
Basic record searching and reporting
The system uses file-handling and data-processing techniques to store and retrieve structured business data.
This project demonstrates the practical implementation of Python functions, CSV file management, user input handling, and basic data analysis using pandas.

 Key Features

- Record daily shift sales
- Store employee working hours
- Rack expenditures
- Pend data to a CSV dashboard file
- Display stored accounting records
- Search for specific entries
- Menu-driven command-line interface

Technologies Used
- Python 3
- Pandas
- CSV File Handling
- Command-Line Interface (CLI)
